    มี 2 ประเภท คือ 1. Hypervisor type 1 จะทำงานบน hardware โดยตรง
                  2. Hypervisor type 2 จะทำงานบน software 
    emulator จะใช้ Hypervisor type 2 เนื่องจาก emulator จะทำงานโดยใช้ hardware ผ่าน software บน computer
